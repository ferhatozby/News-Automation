Telegram News Notification Bot

This Python project I’ve been working on automatically collects news headlines and links from selected sources and sends them as notifications via Telegram.

Key features I focused on:

Preventing duplicate news by checking links

Automatically splitting messages to stay within Telegram’s character limit

Keeping the bot lightweight and easily extendable

Technologies used: Python 3, requests library, and Telegram Bot API.

My goal was to showcase a small but functional automation project to highlight my technical skills and strengthen my LinkedIn portfolio.

As a result, the project’s code remains private on GitHub while the workflow and technical details are visible, making the sharing both secure and professional.
