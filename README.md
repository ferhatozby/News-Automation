# Telegram News Notification Bot

## Project Overview
This is a Python automation project that collects news headlines and links from specified sources and sends them as notifications via Telegram. The bot ensures no duplicate messages are sent and handles message limits automatically.

---

## Features

- Scrapes news headlines and links
- Filters duplicates
- Sends notifications to Telegram
- Handles message length limits
- Lightweight and easy to customize

---

## Technologies Used

- Python 3
- `requests` and `beatifulsoup` library
- Telegram Bot API
